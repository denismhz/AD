Bubblesort for dummies

After one iteration the greatest element in the array is at the last positon and in the right order. For the next iteration we bubblesort the unordered part of the array.

Insertionsort for dummies

Choose an Element(New Elem) out of the array you want to "insert". Compare it to previous Elements in the array. If New Elem is smaller than a previous Element swap them.

Selectionsort for dummies

"Search" the largest element in the array then swap it with the last element in the array. The last element is in right sorted after one iteration. Sort the remaining elements.(largest to end, second largest to end-1, third largest to end-2, ...)

Shellsort for dummies

Compare elements using a gap (vorsortieren, pre-sort with gap) that gets smaller. After each iteration the array is "more sorted" than before. In the end the gap equals 1.

Countingsort for dummies

Temporary Array

Only works with non-negative discrete values. Does not use comparison. Count how many times a value can be found in an array. Save the ocurrences of the values in an temporary array of the size range of values. Save occurences in increasing order lowest to highest value. Write the values back into the array.
